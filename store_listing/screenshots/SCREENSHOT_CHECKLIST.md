# Screenshot Checklist

Capture screenshots on these devices/sizes for both stores.

## Required Devices

### App Store (iOS)
- 6.9" display (iPhone 16 Pro Max — 1320×2868)
- 6.3" display (iPhone 16 Pro — 1206×2622)
- 12.9" iPad Pro (2048×2732) — if supporting iPad

### Play Store (Android)
- Phone: 1080×1920 minimum (16:9 or taller)
- 7" tablet (optional): 1200×1920
- 10" tablet (optional): 1600×2560

## Screens to Capture (in order)

1. **Home screen with profile card** — Shows a filled profile card with name, blood type, allergy badges. Light mode.
2. **QR code display** — Full-screen QR with emergency label and profile summary.
3. **Profile editor** — Filled form showing allergies, medications, emergency contacts sections.
4. **Emergency PDF card** — The print preview showing the credit-card-sized PDF.
5. **QR scanner result** — Scanned profile view showing medical data from a QR code.
6. **Settings screen** — Showing security, backup, and about sections.
7. **Lock screen** — PIN entry screen demonstrating security.
8. **Home screen widget** — Device home screen with the VitalGlyph QR widget.

## Tips
- Use demo data with realistic but fictional medical information
- Capture both light and dark mode variants (pick the best 8)
- Ensure status bar shows full battery and signal for a clean look
- Use `flutter screenshot` or Xcode/Android Studio screenshot tools
